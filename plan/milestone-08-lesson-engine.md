# Milestone 08: Lesson Engine

## Goal

Deliver guided, interactive learning in a Learn Git Branching style.

## Scope

- Lesson DSL
- Validation rules
- Hint system
- Progress tracking

## Deliverables

- First 5 high-value lessons
- Deterministic grading and hint flow

## Exit Criteria

- Scenario tests pass with expected outcomes and feedback keys

## Key Risks

- Content quality becoming the delivery bottleneck

## Suggested First Tasks

1. Define lesson and stage schema.
2. Implement validator contract against snapshots and traces.
3. Implement hint progression levels.
4. Add scenario-based tests for each initial lesson.
