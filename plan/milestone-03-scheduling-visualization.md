# Milestone 03: Scheduling and Visualization

## Goal

Teach CPU virtualization through scheduler behavior.

## Scope

- FIFO scheduler
- Round Robin scheduler
- MLFQ scheduler
- Gantt chart timeline visualization

## Deliverables

- Runtime policy switch
- Metric comparison view (fairness/response/throughput)

## Exit Criteria

- Known workloads produce expected scheduling metrics

## Key Risks

- UI overfitting before engine behavior is stable

## Suggested First Tasks

1. Implement FIFO and RR first, then MLFQ.
2. Create deterministic workload fixtures.
3. Compute scheduling metrics in backend.
4. Render aligned Gantt timeline in frontend.
